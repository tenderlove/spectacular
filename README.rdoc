= spectacular

* http://github.com/tenderlove/spectacular

== DESCRIPTION:

Spectacular is an SNMP client that displays traffic information in the
browser.

== FEATURES/PROBLEMS:

* It updates every second
* I don't know what the units are on the Y axis yet
* X axis is time

== SYNOPSIS:

  $ gem install spectacular
  $ spectacular 10.0.1.1

Open your browser and {click this link}[http://localhost:8080]. If you want,
you can supply the names of interfaces to filter what appears in the browser.

== REQUIREMENTS:

* A router with SNMP enabled

== INSTALL:

* gem install spectacular
